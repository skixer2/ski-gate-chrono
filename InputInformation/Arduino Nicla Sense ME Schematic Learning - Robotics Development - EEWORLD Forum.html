Arduino Nicla Sense ME Schematic Learning [Copy link]

Last edited by Xu Dachui on 2025-12-3 15:43 The Arduino Nicla Sense ME (development board code ABX00050) is a very compact development board designed for the Internet of Things (IoT) and edge computing, integrating a high-performance Bosch sensor array and Bluetooth functionality. Before developing with the Arduino Nicla Sense ME, let's study its schematic to see how the functions of this small development board are implemented at the hardware level. ### Page 1: TOP (Top-level Architecture Diagram) This page is the overview diagram of the entire circuit. It uses green squares to represent sub-circuit modules (Sheets) and shows the main signal connections between modules.

  • Module Division: * Main Processor (ANNA-B112): Core processing unit. * Power: Power management unit. * USB Bridge (SAMD11): Bridge chip for USB communication and debugging. * Sensors: Sensor cluster. * Connectors & Level Shifters: Physical interface and level shifting. * Signal Flow: You can see GPIO, I2C (SDA/SCL), SPI (COPI/CIPO/CLK), and JTAG debug signals moving between modules. The core logic voltage is +1V8 (1.8V), which is crucial for low-power design. --- ### Page 2: CONNECTORS

    Snipaste_2025-12-03_15-00-50.jpg (154.02 KB, downloads: 0)

    This page defines the physical interface between the board and the outside world. * J7 (USB CONNECTOR): Micro-USB or USB-C interface (ZX62 in the schematic, usually Micro-USB, but there may be variations in the Nicla series; this provides VBUS 5V power supply and data transfer). Includes D2 (ESD protection diode) to protect the data lines. * J4 (BATTERY CONNECTOR): 3-pin battery connector, supports single-cell lithium batteries. Includes an NTC pin for monitoring battery temperature and ensuring charging safety. * J5 (ESLOV CONNECTOR): A dedicated Arduino interface (JST 5-pin) for connecting I2C peripherals. Supports 5V input, but the signal has been level-converted. * J1, J2 (Pin Headers): The outer frame is drawn according to the board shape, with pins corresponding one-to-one. Provides VDDIO_EXT, SPI, I2C, and some GPIOs for user expansion. I couldn't find any direct mounting options for the J4 lithium battery on Taobao, so the following connection method can be used:

--- ### Page 3: LEVEL SHIFTERS (Level Conversion)

Since the board's internal core voltage is 1.8V, while external devices typically use 3.3V, a level converter is needed. * U11 (TXB0108): 8-bit bidirectional level converter. Converts internal 1.8V GPIO and SPI signals to the external voltage domain (VDDIO_EXT). * U12 (NTS0304): 4-bit converter, primarily responsible for level conversion between I2C (SDA/SCL) and ESLOC interfaces. * U10 (SN74LVC1T45): Unit converter, used for specific control signal direction conversion. * Important Note: This means that although the internal chip is fragile (1.8V), the pins can be adapted to externally set voltages through these converters. --- ### Page 4: POWER (Power Management)

Snipaste_2025-12-03_15-15-29.jpg (112.92 KB, downloads: 0)

This is the heart of the board, responsible for power supply and battery charging. * U9 (BQ25120): Texas Instruments (TI) highly integrated power management chip (PMIC). * Functions: 1. Linear Charging: Manages lithium battery charging (from USB or VIN). 2. Buck Converter: Reduces battery voltage or USB voltage to +1V8 to power all major chips on the board (MCU, sensors). 3. LDO: Provides auxiliary voltage. * Control: Communicates with the main MCU via the I2C0 bus; the MCU can dynamically adjust the output voltage or monitor battery status. * Circuit Details: * L1 is the inductor required for the buck converter. * D3, D4 are Schottky diodes used for power path selection and protection. --- ### Page 5: USB BRIDGE

Snipaste_2025-12-03_15-28-32.jpg (91.67 KB, downloads: 0)

  • U1 (ATSAMD11D14A): This is a small ARM Cortex-M0+ microcontroller. * Function: It does not run the user's main program, but is used as a USB to serial port/debugger. It allows the computer to recognize the development board and burn the program into the main controller (ANNA-B112). * U14 (AP2112K-3.3): A 3.3V LDO regulator specifically for powering the USB section and SAMD11. * U13 (NTS0304): This level shifter is used to convert the UART signals (RX/TX) of the SAMD11 (3.3V) into 1.8V signals that the ANNA-B112 can accept. --- ### Page 6: SENSORS (Sensors - Core Highlights)

    Snipaste_2025-12-03_15-28-32.jpg (91.67 KB, downloads: 0)

    download attach save to album

    2025-12-3 15:28 上传

    This is the core part of the Nicla Sense ME, integrating high-end sensors from Bosch Sensortec. * U5 (BHI260AP): Smart Sensor Hub + IMU. It's not just a 6-axis IMU (accelerometer + gyroscope); it also integrates a coprocessor called the Fuuser Core. It acts as the SPI master, managing other sensors on the board and relieving the computational burden on the main MCU. * U2 (MX25R1635F): 2MB SPI Flash, dedicated to the BHI260, used to store sensor firmware algorithms or buffered data. * U4 (BMM150): Three-axis magnetometer (compass). * U3 (BMP390L): High-precision barometric pressure sensor (for altitude measurement). * U6 (BME688): Environmental sensor. It can measure gases (VOCs), pressure, humidity, and temperature, and features AI gas recognition capabilities. * Summary: This is a very powerful environmental and motion sensing solution. All data can be processed by the BHI260 before being transmitted to the main controller. --- ### Page 7: ANNAB112 (Main Controller Module)

    Snipaste_2025-12-03_15-29-14.jpg (195.08 KB, downloads: 0)

    download attach save to album

    2025-12-3 15:29 上传

    This is where the user-written code runs. * MD1 (ANNA-B112):This is a Bluetooth module from u-blox. * Internal core is a Nordic nRF52832 (ARM Cortex-M4). * Supports Bluetooth Low Energy (BLE 5.0). * Operating voltage: 1.8V. * U7 (MX25R1635F): Second Flash Chip (2MB). This chip is specifically connected to the ANNA-B112 and is used to store user data, code (Data Logging Flash), or Bluetooth pairing information. * U8 (IS31FL3194): I2C interface RGB LED driver chip. * DL1: RGB LED light, with color and breathing effects controlled by the driver chip, instead of directly occupying the MCU's PWM pin. * Y1: 32.768kHz crystal oscillator, used for precise timing and RTC (Real-Time Clock) for Bluetooth Low Energy. The Arduino Nicla Sense ME boasts an ingenious design: 1. Low-voltage architecture: The entire core operates at 1.8V, significantly reducing power consumption and making it suitable for battery power. 2. Dual-processor architecture: The nRF52832 (ANNA-B112) handles Bluetooth and user logic, while the BHI260 (built-in core) efficiently handles complex sensor data fusion algorithms. 3. Rich sensing capabilities: It integrates a full range of sensors, including motion, magnetism, barometric pressure, temperature, humidity, and gas sensors. 4. Modular power supply: The BQ25120 provides comprehensive battery management. Pinout diagrams are attached:
